# Schedle

Schedle is a game that challenges players to create a schedule for a given set of tasks, in the way an OS scheduler would. It is designed to teach people about scheduling algorithms. 

The main mode is an infinite mode, where a player is given a set of tasks, that arrive at different times, and has to schedule them according to a given algorithm.

There is also a daily challenge mode. 

## Algorithms
- FIFO
- Rate Monotonic
- Earliest Deadline First
